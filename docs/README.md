# Документація для Дизайнерів Контенту (NMTales Game Docs) 🎮

Ласкаво просимо! Ця директорія містить інтерактивні посібники для геймдизайнерів та сценаристів проекту NMTales. Вони розроблені для швидкого проектування, валідації та інтеграції ігрового контенту (діалогів та квестів).

## Склад посібників

1. **[dialogue_designer_guide.html](file:///d:/Development/GameDev/NMTales/game/docs/dialogue_designer_guide.html)**
   * **Суть**: Посібник зі створення розгалужених діалогів. Описує формат JSON-файлів діалогів, структуру реплік, умовні розгалуження (перевірка наявності квестів, виконання умов) та виклик тригерів (наприклад, відкриття вікна тесту чи видача квесту).
   * **Інтерактивні інструменти**: Містить інтерактивний редактор/валідатор діалогів безпосередньо у браузері, який перевіряє правильність зв'язків між вузлами та відповідність JSON-схемі.

2. **[quest_designer_guide.html](file:///d:/Development/GameDev/NMTales/game/docs/quest_designer_guide.html)**
   * **Суть**: Інструкція зі створення ігрових квестів. Описує типи квестів (вбити монстрів, поговорити з NPC, знайти локацію), структуру файлів у форматі JSON та правила нарахування нагород (XP).
   * **Інтерактивні інструменти**: Містить візуальний валідатор структури квестів, що допомагає уникнути помилок перед додаванням файлу до рушія Godot чи backend-сервера.

---

## Як цим користуватися?

Оскільки обидва гайди є самодостатніми HTML-сторінками з інтерактивними скриптами:

1. **Відкрийте файл у будь-якому браузері** (Chrome, Firefox, Edge тощо). Ви можете просто двічі клацнути на файл `dialogue_designer_guide.html` або `quest_designer_guide.html` у провіднику Windows.
2. **Використовуйте інтерактивну форму**:
   * Переглядайте приклади JSON структур.
   * Копіюйте та змінюйте шаблони.
   * Користуйтеся вбудованими інструментами перевірки (валідації) вашого JSON-коду на наявність помилок перед збереженням у гру.
