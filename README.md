# 🎮 NMTales

> Перетвори підготовку до іспитів на захопливий квест! 🚀
> **NMTales** — це безкоштовна інтерактивна гейміфікована вебплатформа, створена для учнів 10–11 класів, які готуються до Національного мультипредметного тесту (НМТ).

---

## ✨ Про проєкт

Підготовка до іспитів не має бути нудною. **NMTales** поєднує навчальний контент із класичними ігровими механіками: очки досвіду (XP), рівні, квести, а також **природне спілкування з NPC завдяки інтеграції великих мовних моделей (LLM)**. Вирішуй реальні тестові завдання, спілкуйся з персонажами, підвищуй свій рівень, отримуй досягнення та опановуй програму НМТ прямо під час гри!

---

## ⚡ Інструкція для швидкого запуску

Виконай ці прості кроки, щоб налаштувати локальний сервер бекенду та запустити гру в Godot на своєму комп'ютері.

### 📋 Системні вимоги

Переконайся, що у тебе встановлено:
* [**.NET 8.0 SDK**](https://dotnet.microsoft.com/download/dotnet/8.0)
* [**Godot Engine (v4.6.2)**](https://godotengine.org/download) (підійде стандартна версія)

---

## 🚀 Налаштування та запуск

### 1️⃣ Клонування репозиторію
Завантаж проєкт на свій комп'ютер (бажано уникати кирилиці та пробілів у шляху до папки, щоб запобігти помилкам зчитування):
```bash
git clone https://github.com/<your-username>/NMTales.git
cd NMTales/game
```

---

### 2️⃣ Запуск API-бекенду
Для роботи гри необхідний запущений бекенд, який керує авторизацією, збереженням прогресу та базою питань.

1. Відкрий термінал та перейди до папки бекенду:
   ```bash
   cd backend/NMTales.Backend
   ```
2. Віднови залежності та запусти сервер:
   ```bash
   dotnet run
   ```
   *Тепер бекенд працює за адресою `http://localhost:5142` (інтерактивна документація Swagger доступна на `http://localhost:5142/swagger`). Залиш цей термінал відкритим!*

---

### 3️⃣ Запуск клієнта гри
Тепер відкриємо гру в ігровому рушії Godot:

1. Відкрий **Godot Engine (v4.6.2)**.
2. Натисни кнопку **Import** (Імпорт) та обери файл `project.godot`, що лежить у папці `game/`.
3. Після імпорту та відкриття проєкту в редакторі натисни клавішу **F5** (або кнопку Play у верхньому правому куті), щоб запустити гру!
4. Створи акаунт (знадобляться лише юзернейм та пароль) та починай грати!

---

## 🛠️ Структура проєкту

* **`/backend`**: Серверна частина на ASP.NET Core Web API (знаходиться в `game/backend/NMTales.Backend`).
* **`/ui`**: Інтерфейси гри, макети меню та сцени Godot.
* **`/assets`**: Текстури, звуки, шрифти та графічні компоненти.
* **`/locations` та `/parts`**: Карти рівнів Godot, ігрові зони та модульні асети.
