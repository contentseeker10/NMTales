# NMTales
Безкоштовна веб ігрова платформа для підготовки до Національного Мультипредметного Тесту (НМТ) для учнів 10-11 класів

## Запуск

Для запуску поточної версії: 
- завантажте файли репозиторію в зручну директорію **(бажано без кирилиці та пробілів в шляху)**
- завантажте *Godot Engine версії 4.6.2* **(free)** 
- запустіть Godot
- імпортуйте проект з обраної директорії
- F5 to play
